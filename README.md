# 🩸 BloodCare+

### Smart Blood Donation & Request Management System

---

## 🚀 Overview

BloodCare+ is a web-based platform that helps connect **blood donors** with **people in need** quickly and efficiently.
It simplifies the process of finding available donors using a centralized system.

---

## 🎯 Features

- 👤 Donor Registration
- 🔍 Search Donors by Blood Group & Location
- 📊 Real-Time Availability Tracking
- 🧑‍🤝‍🧑 Volunteer/Donor Management
- 🔐 Secure Login System
- 📱 Responsive User Interface

---

## 🛠️ Tech Stack

### 💻 Frontend

- HTML5 🧱
- CSS3 🎨
- Bootstrap ⚡
- Jinja Templates 🧩

### ⚙️ Backend

- Python 🐍
- Flask 🌐

### 🗄️ Database

- SQLite / MySQL 💾

---

## 🔄 How It Works

1. 👤 Users register as donors with their details (blood group, location, contact).
2. 🔎 Patients search for required blood group.
3. 📋 System filters and displays matching donors.
4. 📞 Users contact donors directly.
5. 📊 Admin can manage users and monitor activity.

---

## 🧠 Project Workflow

- User Input ➝ Data Stored in Database
- Search Query ➝ Backend Processing (Flask)
- Matching Logic ➝ Display Results (Frontend)
